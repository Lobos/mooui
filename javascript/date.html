MooUI.DatePicker mooui-datepicker.js

日期选择插件。

Syntax:

var dp = new MooUI.DatePicker(options);

Options:

  1. bind - (string 或者 elements) css3选择器，或者元素数组。

Methods:bind

增加绑定元素

Syntax:

openbox.bind(inputs)

Arguments:

  1. inputs - (string 或者 elements) css3选择器，或者元素数组。

Return:

  • 对象自身。

Example

<form id="date_form_1" class="form-horizontal">
    <input type="text" class="date" />
    <input type="text" class="date" />
</form>

var dp = new MooUI.DatePicker({
    bind: '#date_form_1 .date'
});

var new_date = new Element('input.date', {type:'text'}).inject('date_form_1');
dp.bind(new_date);

new Element('input.date', {type:'text'}).inject('date_form_1').datepicker();
