MooUI.ProgressBar mooui-progressbar.js

进度条。

Syntax:

var progress = new MooUI.ProgressBar(element, options);

Arguments:

  1. element - (string 或者 element) 进度条css3选择器或者元素。
  2. options - (object)

Options:

  1. barClass - (string) 内滑动条类名。默认值： bar

Methods:set

设置当前值。

Syntax:

progress.set(to)

Arguments:

  1. to - (int) 设定的位置值，单位%

Return:

  • 对象自身。

Methods:start

移动到设定值。

Syntax:

progress.start(to, total)

Arguments:

  1. to - (int) 移动到的位置值，如果total为空，单位为%。如果total有数值，无单位。
  2. total - (int 可为空) 如果不为空，移动位置为 to/total的百分比。

Return:

  • 对象自身。

Methods:imitation

模拟进度条。

Syntax:

progress.imitation(total)

Arguments:

  1. total - (int) 用来控制模拟速度。

Return:

  • 对象自身。

Example

进度

<div id="progress_1" class="progress progress-striped"></div>
<button class="btn btn-primary">测试</button>
<span></span>

var progress_1 = new MooUI.ProgressBar('progress_1', {
    onComplete: function () {
        $('progress_1').getNext('span').set('html', 'done');
    }
});
$('progress_1').getNext('.btn').addEvent('click', function () {
    progress_1.set(0).imitation(100);
});

设置进度

<div id="progress_2" class="progress progress-striped"></div>
<button class="btn btn-primary" rel="20">20%</button>
<button class="btn btn-success" rel="60">60%</button>
<button class="btn btn-warning" rel="100">100%</button>

var progress_2 = new MooUI.ProgressBar('progress_2');
$('progress_2').getAllNext('.btn').addEvent('click', function () {
    progress_2.set(this.get('rel'));
});
