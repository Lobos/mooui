MooUI.Openbox mooui-openbox.js

弹出窗口插件。

Syntax:

var openbox = new MooUI.Openbox(options);

Options:

  1. request - (object) 动态获取数据的参数。和Request.JSON.options相同。
  2. pageAble - (bool) 是否分页。默认值： true
  3. filterAble - (bool) 是否显示筛选。默认值： false
  4. pagination - (object) MooUI.Pagination.options。
  5. header - (json 必填) 见下。

Example

基础弹出窗口

这是一个弹出窗体。
<button id="btn_1" class="btn btn-primary">弹出</button>
<div style="padding: 20px">这是一个弹出窗体。</div>

var openbox_1 = new MooUI.Openbox({
    title: '一个弹出窗口',
    width: 400,
    height: 240,
    destroyOnClose: false,
    content: $('btn_1').getNext()
});
$('btn_1').addEvent('click', function () {
    openbox_1.open();
});
