MooUI.Openbox mooui-openbox.js

弹出窗口插件。

Syntax:

var openbox = new MooUI.Openbox(options);

Options:

  1. id - (string, 可为空) openbox的id，为空时自动创建。
  2. container - (string 或者 element, 可为空) openbox的外部容器，默认为document.body。
  3. title - (string) 头部标题。
  4. content - (string 或者 element) 内容。
  5. pad - (int) 距离屏幕的最小边距。（双边合计值），默认值： 100
  6. width - (int 或 'auto') 弹出窗口宽度，当值为 auto时，自动适应内部内容。默认值： auto
  7. height - (int 或 'auto') 弹出窗口高度（不包含标题和底部按钮），当值为 auto时，自动适应内部内容。默认值： auto
  8. opacity - (0-1小数) 窗口透明度，默认值： 1
  9. overlayOpacity - (0-1小数) 背景遮罩层透明度，当值为 0时，不设置遮罩层。默认值： 0.1
  10. closeOnOverlayClick - (bool) 点击背景遮罩层时，关闭窗口。默认值： true
  11. destroyOnClose - (bool) 关闭时是否销毁整个对象。默认值： false
  12. arise - (bool) 点击窗口时，移动到最上层。默认值： false
  13. fixed - (bool) 窗口位置是否固定定位（为false时使用absolute）。默认值： true
  14. maskAll - (bool) 使用内部遮罩时是否包含标题栏和按钮。默认值： true
  15. showTitle - (bool) 是否显示标题。默认值： true
  16. showClose - (bool) 是否显示关闭按钮。默认值： true
  17. dragAble - (bool) 是否可以拖动。默认值： false
  18. resizeAble - (bool) 是否可以拖动改变尺寸。默认值： false
  19. position - (object 可为空) 窗口显示位置，参考element.style。默认为空，窗口显示在屏幕正中。
  20. duration - (int) 窗口显示/隐藏动画执行时间。默认值： 200
  21. css - (object) 窗口元素类名。
  22. buttons - (array)
    • text - (string) 按钮文字。
    • events - (function 或 object) 当值为function时，添加click事件。为空时，添加关闭窗口事件。
    • type - (string) 按钮样式。可选值为 primary|success|info|danger|warning|inverse

Events:

  1. close(data) - 窗口关闭时触发事件。

Methods:open

打开窗口

Syntax:

openbox.open()

Return:

  • 对象自身。

Methods:close

关闭窗口

Syntax:

openbox.close(data)

Arguments:

  1. data - 任意格式。供 close事件调用。

Return:

  • 对象自身。

Methods:arise

窗口移动到最上层

Syntax:

openbox.arise()

Return:

  • 对象自身。

Methods:destroy

销毁对象

Syntax:

openbox.destroy()

Return:

  • 无

Methods:lock

窗口设置移动或改变大小时，锁定窗口。

Syntax:

openbox.lock()

Return:

  • 对象自身

Methods:unlock

取消锁定。

Syntax:

openbox.unlock()

Return:

  • 对象自身

Methods:set

设置。

Syntax:

openbox.set(property, content)

Arguments:

  1. property - (string) 可选值 title|content。
  2. content - (string 或者 element) 同options.content。

Return:

  • 对象自身

Example

基础弹出窗口

<button id="btn_1" class="btn btn-primary">弹出</button>

var openbox_1 = new MooUI.Openbox({
    title: '一个弹出窗口',
    width: 400,
    height: 200,
    destroyOnClose: false,
    content: '一个弹出的窗口',
    buttons: [
        { text: '测试', events: function () { alert('test'); }, type: 'primary' },
        { text: '关闭' }
    ]
});
$('btn_1').addEvent('click', function () {
    openbox_1.open();
});

拖动和改变尺寸

这是一个可以拖动的窗体。
<button id="btn_2" class="btn btn-primary">弹出</button>
<button class="btn btn-inverse">锁定</button>
<button class="btn btn-success">取消锁定</button>
<div class="openbox-content">这是一个可以拖动的窗体。</div>

var openbox_2 = new MooUI.Openbox({
    title: '我可以拖动',
    width: 400,
    height: 240,
    overlayOpacity: 0,
    destroyOnClose: false,
    dragAble: true,
    resizeAble: true,
    content: $('btn_2').getNext('div')
});
$('btn_2').addEvent('click', function () {
    openbox_2.open();
});
$('btn_2').getNext('.btn-inverse').addEvent('click', function () {
    openbox_2.lock().set('title', '我被锁住了');
});
$('btn_2').getNext('.btn-success').addEvent('click', function () {
    openbox_2.unlock().set('title', '我可以拖动');
});

MooUI.Openbox.Request mooui-openbox.js

继承于 MooUI.Openbox。ajax请求数据。

Syntax:

var openbox = new MooUI.Openbox.Request(options);

Options:

  1. destroyOnClose - 默认值为 true
  2. request - (object) 参见 Request.HTML.options。

Methods:load

读取数据

Syntax:

openbox.load(options)

Arguments:

  1. options - (object) 参见 Request.HTML.options。

Return:

  • 对象自身

Example

<button id="btn_3" class="btn btn-primary">弹出</button>

$('btn_3').addEvent('click', function () {
    var openbox_3 = new MooUI.Openbox.Request({
        title: '读取中...',
        request: {
            method: 'get',
            url: 'json/openbox/page1.html'
        }
    });
    openbox_3.open();
});
