Dropdowns moodropdown.js

下拉菜单（dropdown menu）插件

Syntax:

var dd = new MooDropdown(el[, options]);

Arguments:

  1. el - 单个或一组element 或者 css3选择器。
  2. options - 见下面。

Options:

  1. reveal - (bool) 值为true时，菜单使用 Fx.Reveal 动画显示/隐藏，需要 Fx.Reveal 支持。默认值为 false
  2. dropbox - (string 或者 element) 下拉菜单，css3选择器(相对el)或者Element。默认值为 .dropdown-menu
  3. revealClass - (string) 如果el有revealClass这个class，无需设置 reveal: true。默认值为 dropdown-reveal
  4. activeClass - (string) 显示/隐藏时添加/移除的class。默认值为 active

Return:

  • (object) 一个新 MooDropdown 对象

Methods:

  1. open - 显示下拉菜单
  2. close - 隐藏下拉菜单
