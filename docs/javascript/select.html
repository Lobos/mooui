MooUI.Select mooui-select.js

一个模拟select元素的插件。

Syntax:

var select = new MooUI.Select(container[, options]);

Arguments:

  1. container - (string 或者 element) 需要转换为select的元素，css3选择器或者元素。
  2. options - 见下面。

Options:

  1. name - (string) 类似input.name属性，表单提交时，可以通过name取值。
  2. boxWrapper - (string 或者 element) 下拉选择项容器，默认为document.body。
  3. maxHeight - (int 单位:px) 下拉选择框的最大高度，默认值： 200。
  4. borderFix - (int 单位:px) 下拉选择框偏移修正，默认值： 1。
  5. changeValue - (bool) 选项改变时是否改变容器的值，默认值： true。（注：设计这个选项是为了动态dropdown）
  6. request - (options) Request.JSON 的options。
  7. styleClass - (options) 内部元素类名。
    • box - 下拉框。默认值：select-box
    • item - 下拉框选项。select-item
    • inner - 内部元素。select-inner
  8. noneData - (object 可选) 空值。
  9. validate - (string 可选) 参考 MooUI.Validator 数据项。
  10. data - (Array[object] 可选) 下拉选项，详见数据结构。

Data Structure：

[{text:'', 'class':'', value:'', events:{}}]
  1. text - 选项显示文字。
  2. class - 选项类属性。同element。
  3. value - 选项值。
  4. events - 事件。同element。

Return:

  • (object) 一个新 MooUI.Select 对象
