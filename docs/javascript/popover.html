MooUI.Popover mooui-popover.js

弹出提示信息

Syntax:

var dd = new MooUI.Popover(el[, options]);

Arguments:

  1. el - 单个或一组element 或者 css3选择器。
  2. options - 见下面。

Options:

  1. reveal - (bool) 值为true时，菜单使用 Fx.Reveal 动画显示/隐藏，需要 Fx.Reveal 支持。默认值为 false
  2. dropbox - (string 或者 element) 下拉菜单，css3选择器(相对el)或者Element。默认值为 .dropdown-menu
  3. revealClass - (string) 如果el有revealClass这个class，无需设置 reveal: true。默认值为 dropdown-reveal
  4. activeClass - (string) 显示/隐藏时添加/移除的class。默认值为 active

Return:

  • (object) 一个新 Popover 对象

Methods:

  1. show - 显示弹出信息
  2. destroy - 隐藏下拉菜单

Examples


<form id="form_popover" class="form-inline">
    <input type="text" value="这是一个弹出的提示信息。" /><button type="button" class="btn">Submit</button>
    <br />
    <label><input type="radio" name="placement" checked="checked" value="top" /> 上</label>
    <label><input type="radio" name="placement" value="bottom" /> 下</label>
    <label><input type="radio" name="placement" value="left" /> 左</label>
    <label><input type="radio" name="placement" value="right" /> 右</label>
</form>

var btn = $$('#form_popover .btn')[0];
var popover = new MooUI.Popover(btn, { destroyOnClose: true });
btn.addEvent('click', function () {
    popover.close();
    var placement = $$('#form_popover input:checked')[0].get('value');
    popover.show(btn.getPrevious().get('value'), {
        placement: placement
    });
});
