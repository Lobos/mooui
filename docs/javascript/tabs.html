MooTabs mootabs.js

一个简单的标签切换插件。

Syntax:

var fv = new MooTabs(tabs, slides[, options]);

Arguments:

  1. tabs - (string 或者 elements) 标签，css3选择器或者元素数组。
  2. slides - (string 或者 elements) 内容项，css3选择器或者元素数组。和标签对应。
  3. options - 见下面。

Options:

  1. activeClass - (string) 标签显示/隐藏时增加的class名。默认值为 active
  2. slideClass - (string) ajax动态添加的内容容器类名。
  3. lazyLoad - (bool) 为true时，如果标签内容为ajax获取，将会延时到显示时才获取。为 flase时，在tabs创建时获取。默认值为 true

Return:

  • (object) 一个新 MooTabs 对象

Methods:addTab

动态添加一个标签。

Syntax:

fv.addTab(tab, slide)

Arguments:

  1. tab - (element) 标签元素。
  2. slide - (element) 可为null。为null时，通过 tab.href 获取ajax内容。

Return:

  • (object) 对象本身。
