MooUI.Pagination mooui-pagination.js

一个分页插件。

Syntax:

var page = new MooUI.Pagination(container[, options]);

Arguments:

  1. container - (string 或者 element) page容器，css3选择器或者元素。
  2. options - 见下面。

Options:

  1. size - (int) 每页数量。默认值： 10
  2. index - (int) 当前显示页。默认值： 1
  3. total - (int) 总数。
  4. capacity - (int) 显示的页数范围。默认值： 10
  5. showCount - (bool) 是否显示统计信息。默认值： false

Return:

  • (object) 一个新 MooUI.Pagination 对象

Events:

  1. pageChange(index) - 分页按钮点击触发事件。index 点击页面页数。

Methods:change

页码改变

Syntax:

sel.change(options)

Arguments

  1. options - (object 或者 int) 值为object时，等同Options。值为int时，表示改变的页码。

Return:

  • MooUI.Pagination 自己

Methods:draw

重绘页码

Syntax:

sel.draw([index])

Arguments

  1. index - (int 可为空) 显示的中心页。如果为空，以当前的选中页作为中心页。

Return:

  • MooUI.Pagination 自己

Example

<div id="pager_1" class="pagination"></div>

new MooUI.Pagination('pager_1', {
    index: 2,
    total: 1008,
    onPageChange: function (index) {
        this.change(index);
    }
}).draw();
