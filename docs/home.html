简单的说明

这是一个在 bootstrap基础上修改的UI框架。 GitHub地址 >>
没有bootstrap那么强大的css生成器，如果需要修改，推荐安装 nodejs下的less模块生成css文件，使用uglify-js模块压缩js文件。

  1. 首先到 node.js的官网下载node的最新版本，安装。
  2. 使用 npm 安装uglify-js和less，可能会比较慢。 
    npm install -g uglify-js less
  3. 如果使用less的时候遇到错误，建议使用 less的github源文件 替换掉npm安装目录下的less文件。

主要区别的地方

  • 稍微偏向metro风格，类名称基本保留 bootstrap 原版。
  • 图标替换为 font-awesome。
  • javascript使用了mootools框架，而不是jquery，基本是我自己写的插件，功能和 bootstrap 原版的差异很大。
